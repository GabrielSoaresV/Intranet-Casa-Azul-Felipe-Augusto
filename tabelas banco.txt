1. Tabela ASSOCIACAO

ID → BIGINT, PK, auto increment

ID_AVALIACAO → BIGINT, FK para AVALIACOES(ID_AVALIACAO)

ID_JOVEM → VARCHAR(255), FK para JOVEM_APRENDIZ(MATRICULA)

2. Tabela AVALIACOES

ID_AVALIACAO → BIGINT, PK

AVALIACAO → VARCHAR(255)

DATA_AVALIACAO → DATE

NUMERO_AVALIACAO → INT

3. Tabela EMPRESA

CNPJ → VARCHAR(255), PK

RH_NOME_RESPONSAVEL → VARCHAR(255)

RH_EMAIL_RESPONSAVEL → VARCHAR(255)

EMAIL_EMPRESA → VARCHAR(255)

NOME_EMPRESA → VARCHAR(255)

TELEFONE_EMPRESA → VARCHAR(255)

4. Tabela JOVEM_APRENDIZ

MATRICULA → VARCHAR(255), PK

CONTRATACAO → DATE

EMPRESA → VARCHAR(255), FK para EMPRESA(CNPJ)

NOME → VARCHAR(255)

PERIODO_AVALIACAO → VARCHAR(255)

RESCISAO → DATE

EMAIL → VARCHAR(255)

TELEFONE → VARCHAR(255)

NOMERESPONSAVEL → VARCHAR(255)

TELEFONERESPONSAVEL → VARCHAR(255)

OBSERVACOES → VARCHAR(255)

Relacionamentos

ASSOCIACAO.ID_AVALIACAO → AVALIACOES.ID_AVALIACAO

ASSOCIACAO.ID_JOVEM → JOVEM_APRENDIZ.MATRICULA

JOVEM_APRENDIZ.EMPRESA → EMPRESA.CNPJ
